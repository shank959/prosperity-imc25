Dashboard Guide

Loading Data
Separate CSV files (no bot runs yet): Drop or select prices_round_N_day_X.csv into Prices (select multiple days at once). Drop trades_round_N_day_X.csv into Trades. These are market-history files from IMC — no own-trade markers will appear.
output.log (after running bot): Drop the single output.log into Log. This auto-loads OB data, market trades, your bot's quotes (★/◆), own fills (× take, ○ make, ◆ clear), and print output in the Log Viewer.
Chart Navigation
Scroll wheel — zoom in/out horizontally (Y auto-fits)
Shift + drag — pan left/right
Box drag — zoom to selection
Double-click — reset to full view
Chart Layers
━ Blue lines — bid prices (L1 solid, L2/L3 faded)
━ Red lines — ask prices (L1 solid, L2/L3 faded)
- - Green dashed — fair value (model selectable via FV dropdown)
━ White line — mid price
★ Stars — your make quotes (blue=bid, orange=ask)
◆ Diamonds — your clear quotes (teal=bid, magenta=ask)
Own fills:
TB/TS × — take fills (yellow=ask take, purple=bid take)
MB/MS ★ — make fills (purple=buy, orange=sell)
CB/CS ◆ — clear fills (green both sides)
Bot trades:
M □ purple square — inside-spread trade
S/B ▲ — taker trades (small ≤5, big >5)
Fair Value Model (Extensible)
Use the FV dropdown to switch fair-value methods for dynamic products: OuterWallMid, EMA(OuterWallMid), or Raw Mid. EMA(OuterWallMid) smooths jagged wall-mid jumps from one-sided books. EMAα controls smoothing (higher=faster, lower=smoother).
IMC Official JSON (IMC site output)
After submitting your bot to the IMC Prosperity website, download the result JSON (e.g. 72288.json). Drop it into IMC. Loads OB data, trades, and print logs. Also adds: total profit in quick stats, final positions per product, and a pre-sampled PnL curve (graphLog). Note: official format does NOT include trader names (only "" or "SUBMISSION") — the Who filter won't have names.
Trader Filter & Large Trade Highlighting
Who filter (Trades tab): after loading a local output.log, select a specific bot/trader to see only their trades and highlight them in yellow on the main chart.
Big≥ (controls bar): set a minimum qty threshold — trades at or above this size get enlarged markers with a glow on the main chart. Use to spot large informed trades.
Analytics Tabs
Spread — bid-ask spread over time + distribution stats. P90 = safe quote width.
Returns — autocorrelation (negative lag-1 = mean-reverting!), return distribution
Bots — trade-size signatures (bot fingerprints), daily range, price impact
MM Opt — grid-search take_width × make_width, find best simulated PnL
Strategy — PnL slope, position sawtooth health, fill edge scatter, spread zones, qty distribution
